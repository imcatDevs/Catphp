Captcha

Cat\Captcha — 이미지/수학 캡차
captcha()

GD 기반 이미지 캡차 및 수학 캡차를 제공합니다. 봇 자동 가입/스팸을 방지합니다.

세션에 정답을 저장하고, verify()로 사용자 입력을 검증합니다. 대소문자 구분 없이 비교합니다.

전체 메서드 레퍼런스
메서드반환설명
image()never이미지 캡차 직접 출력 (PNG)
src()stringbase64 data URI (img src용)
html(string $id, string $refreshUrl)string캡차 HTML 위젯
math()array수학 캡차 {question, html}
verify(string $input)bool사용자 입력 검증
size(int $width, int $height)self이미지 크기 설정
length(int $n)self코드 길이 설정
이미지 캡차
// 폼에 캡차 삽입
echo '<img src="' . captcha()->src() . '" alt="captcha">';
echo '<input name="captcha" placeholder="위 코드 입력">';

// HTML 위젯 (새로고침 버튼 포함)
echo captcha()->html('captcha', '/api/captcha/refresh');
수학 캡차 + 검증
// 수학 캡차 ("7 + 3 = ?")
$math = captcha()->math();
echo $math['html'];  // 질문 HTML

// POST 핸들러에서 검증
if (!captcha()->verify($_POST['captcha'])) {
    json()->fail('캡차가 일치하지 않습니다');
}
요구사항: ext-gd PHP 확장이 필요합니다. 수학 캡차는 GD 없이도 동작합니다.
관련: Csrf Valid
