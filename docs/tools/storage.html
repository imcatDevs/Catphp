Cat\Storage — 파일시스템 추상화
storage()

로컬/S3 파일시스템 추상화입니다. 디스크별 설정으로 로컬 저장소와 S3 호환 스토리지를 동일한 API로 사용합니다.

disk()로 디스크를 전환하고, put()/get()/delete()로 파일을 관리합니다. stream()으로 대용량 파일도 처리할 수 있습니다.

주요 메서드
메서드반환설명
disk(string $name)self디스크 선택
put(string $path, string $content)bool파일 저장
get(string $path)?string파일 읽기
exists(string $path)bool파일 존재 여부
delete(string $path)bool파일 삭제
copy / movebool파일 복사/이동
size / lastModified / mimeType—파일 메타 정보
url(string $path)string공개 URL 반환
files(string $dir, bool $recursive)array디렉토리 목록
stream(string $path)void파일 스트리밍 (대용량)
append(string $path, string $content)bool파일 추가 쓰기
사용 예제
// 로컬 디스크 (기본)
storage()->put('logs/custom.log', '에러 기록');
$content = storage()->get('logs/custom.log');

// S3 디스크로 전환
storage()->disk('s3')->put('uploads/photo.jpg', $binary);
$url = storage()->disk('s3')->url('uploads/photo.jpg');

// 대용량 파일 스트리밍 다운로드
storage()->stream('exports/large.csv', 'download.csv');
팁: disk('public')은 웹에서 접근 가능한 경로, disk('local')은 비공개 경로입니다. S3 설정은 config/app.php의 storage.disks.s3에서 지정합니다.
