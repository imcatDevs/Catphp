Sanitizer

Cat\Sanitizer — HTML 정제
sanitizer()

HTML 콘텐츠에서 XSS 공격 벡터를 제거하는 보안 도구입니다. WYSIWYG 에디터, 댓글, 게시판 등 사용자 입력 HTML을 안전하게 정제합니다.

DOMDocument 기반 파싱 + 7단계 보안 레이어로 195개 이상의 XSS 공격 패턴을 방어합니다.

전체 메서드 레퍼런스
메서드반환설명
clean(string $html)stringHTML 정제 (기본: 텍스트만 허용)
allowImages(bool $v=true)self<img> 태그 허용
allowLinks(bool $v=true)self<a> 태그 허용
allowMedia(bool $v=true)self<video>, <audio> 허용
allowTables(bool $v=true)self<table> 관련 태그 허용
allowTags(array $tags)self추가 태그 허용 (예: ['div', 'span'])
allowAttrs(array $attrs)self추가 속성 허용 (예: ['class', 'id'])
allowProtocols(array $p)self허용 프로토콜 변경 (기본: http, https, mailto)
기본 사용법
// 기본: 텍스트만 허용
$clean = sanitizer()->clean($userHtml);

// 이미지 + 링크 허용
$clean = sanitizer()->allowImages()->allowLinks()->clean($content);

// 체이닝으로 다양한 태그 허용
$clean = sanitizer()
    ->allowImages()
    ->allowLinks()
    ->allowMedia()
    ->allowTables()
    ->allowTags(['div', 'span', 'p', 'h1', 'h2'])
    ->allowAttrs(['class', 'id'])
    ->clean($html);
방어 범위
위험 태그 제거
  • • script, style, iframe, object
  • • form, input, button, select
  • • meta, link, base
  • • SVG 애니메이션 (animate, set)
  • • template, noscript, slot
위험 속성 제거
  • • on* 이벤트 핸들러 (전체)
  • • style (CSS 공격)
  • • srcdoc, formaction, action
  • • xlink:href, xmlns
  • • autofocus, contenteditable
공격 방어 예시
// 스크립트 태그 → 제거
sanitizer()->clean('<script>alert(1)</script>');
// 결과: ''

// 이벤트 핸들러 → 제거
sanitizer()->allowImages()->clean('<img src=x onerror=alert(1)>');
// 결과: '<img src="x">'

// javascript: URL → href 제거
sanitizer()->allowLinks()->clean('<a href="javascript:alert(1)">X</a>');
// 결과: '<a>X</a>'

// 인코딩 우회 → 디코딩 후 차단
sanitizer()->allowLinks()->clean('<a href="javascript:alert(1)">X</a>');
// 결과: '<a>X</a>'

// DOM Clobbering → 위험 id 제거
sanitizer()->allowImages()->clean('<img id="document" src="x">');
// 결과: '<img src="x">'
7단계 보안 레이어: ① 제어문자/Null 제거 → ② HTML 엔티티 디코딩 → ③ DOMDocument 파싱 → ④ 위험 태그/속성 제거 → ⑤ URL 프로토콜 검증 → ⑥ DOM Clobbering 방어 → ⑦ 정규식 최종 정리
테스트 완료: 195개 XSS 공격 벡터 테스트 통과 (기본 XSS, 인코딩 우회, mXSS, SVG/MathML, DOM Clobbering, Prototype Pollution, CSS 공격 등)
관련: Guard Csrf Security 데모
