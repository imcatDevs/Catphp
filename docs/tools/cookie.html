Cat\Cookie — 보안 쿠키 관리
cookie()

보안 쿠키 관리 도구입니다. 쿠키 값을 Encrypt 도구와 연동하여 자동 암호화/복호화하며, HttpOnly, Secure, SameSite 속성을 기본 적용합니다.

사용자가 브라우저 개발자 도구에서 쿠키 값을 확인해도 암호화된 문자열만 보이므로, 세션 토큰이나 사용자 설정 등 민감한 데이터를 안전하게 저장할 수 있습니다. 배열/객체 값은 자동으로 JSON 변환됩니다.

전체 메서드 레퍼런스
메서드반환설명
get(string $name, mixed $default = null)mixed쿠키 읽기 (자동 복호화)
set(string $name, mixed $value, int $ttl = 86400)bool쿠키 설정 (암호화 + 보안 속성 자동)
del(string $name)bool쿠키 삭제
has(string $name)bool쿠키 존재 여부 확인
기본 사용법
// 쿠키 설정 (7일, 암호화 자동)
cookie()->set('locale', 'ko', 604800);

// 쿠키 읽기 (자동 복호화, 없으면 기본값)
$locale = cookie()->get('locale', 'en');

// 쿠키 존재 확인 + 삭제
if (cookie()->has('locale')) {
    cookie()->del('locale');
}
Geo 연동 — 언어 설정 저장
// 언어 전환 시 쿠키에 저장
cookie()->set('lang', $_GET['lang'] ?? 'ko', 86400 * 365);

// 다음 요청에서 자동 적용
$lang = cookie()->get('lang', 'ko');
geo()->locale($lang);
보안 기본값: HttpOnly: true (JS 접근 차단), Secure: auto (HTTPS에서 자동), SameSite: Lax (CSRF 보호)
관련: Encrypt Geo
