Cat\Search — 전문 검색
search()

SQL LIKE 기반 전문 검색 도구입니다. 여러 테이블·컬럼을 대상으로 검색하고, highlight()로 검색어를 강조 표시합니다.

Paginate 도구와 연동하여 검색 결과 페이지네이션이 가능하며, Cache 도구로 자주 검색되는 쿼리의 결과를 캐싱할 수 있습니다. validateIdentifier()로 테이블·컬럼명 SQL Injection을 차단합니다.

전체 메서드 레퍼런스
메서드반환설명
query(string $keyword)self검색 키워드 설정
in(string $table, string ...$cols)self검색 대상 테이블·컬럼
limit(int $n)self결과 개수 제한
offset(int $n)self결과 시작 위치
results()array검색 결과 배열
count()int전체 결과 수
highlight(string $text)string검색어 <mark> 강조
사용 예제
$keyword = $_GET['q'] ?? '';

$s = search()
    ->query($keyword)
    ->in('posts', 'title', 'body')
    ->in('pages', 'title', 'content')
    ->limit(10)->offset(0);

$results = $s->results();
$total   = $s->count();

// 검색어 강조
foreach ($results as $row) {
    echo $s->highlight($row['title']);
}
보안: 테이블·컬럼명은 validateIdentifier()로 SQL Injection을 방지합니다. 검색어는 prepared statement로 전달되므로 안전합니다.
관련: Paginate Cache DB
