Cat\Feed — RSS/Atom 피드
feed()

RSS 2.0 및 Atom 형식의 XML 피드를 생성합니다. 블로그 게시글, 뉴스, 팟캐스트 등의 콘텐츠를 피드로 발행할 수 있습니다.

각 항목에 <guid isPermaLink="true"> 요소가 자동 포함되어 피드 리더에서 중복을 올바르게 처리합니다. fromQuery()로 DB 결과를 바로 피드 아이템으로 변환할 수 있습니다. 체이닝 메서드는 clone 기반입니다.

전체 메서드 레퍼런스
메서드반환설명
title(string $title)self피드 제목
description(string $desc)self피드 설명
link(string $url)self피드 링크
items(array $items)self피드 아이템 배열 설정
fromQuery(array $rows, ...)selfDB 쿼리 결과를 아이템으로 변환
rss()neverRSS 2.0 출력 (헤더 + echo + exit)
atom()neverAtom 출력 (헤더 + echo + exit)
render(string $format = 'rss')stringXML 문자열 반환 (exit 없음)
DB 연동 — fromQuery()
// DB 결과를 바로 RSS 피드로 출력
$posts = db()->table('posts')
    ->where('status', 'published')
    ->orderBy('created_at', 'DESC')
    ->limit(20)->all();

feed()->title('CatPHP 블로그')
    ->link('https://catphp.dev')
    ->description('PHP 프레임워크 소식')
    ->fromQuery($posts)
    ->rss();  // Content-Type + echo + exit
수동 아이템 + Atom
// items() 배열로 직접 설정
$xml = feed()->title('릴리스 노트')
    ->link('https://catphp.dev/releases')
    ->items([
        ['title' => 'v2.0', 'content' => '새 기능...', 'created_at' => '2024-01-01'],
    ])
    ->render('atom');  // 문자열 반환 (exit 없음)
캐싱: cache()->remember('feed', fn() => feed()->...->render(), 3600) 패턴으로 피드를 캐싱하면 DB 부하를 줄일 수 있습니다.
