Cat\Http — cURL HTTP 클라이언트
http()

cURL 기반 HTTP 클라이언트입니다. GET/POST/PUT/DELETE 요청, JSON 전송, 파일 업로드/다운로드를 체이닝 API로 간결하게 사용할 수 있습니다.

base()로 기본 URL을 설정하면 이후 요청에서 상대 경로만 지정할 수 있습니다. 응답은 Http\Response 객체로 반환되어 status(), body(), json() 등으로 접근합니다.

전체 메서드 레퍼런스
메서드반환설명
base(string $url)self기본 URL 설정
header(string $k, string $v)self요청 헤더 추가
timeout(int $sec)self타임아웃 설정 (초)
get(string $url, array $query)ResponseGET 요청
post(string $url, array $data)ResponsePOST 요청 (form-urlencoded)
put(string $url, array $data)ResponsePUT 요청
delete(string $url)ResponseDELETE 요청
json(string $url, array $data)ResponseJSON POST (Content-Type 자동)
upload(string $url, string $filePath, string $fieldName, array $extra)Response파일 업로드 (multipart)
download(string $url, string $savePath)bool파일 다운로드
Response 메서드
메서드반환설명
status()intHTTP 상태 코드
body()string응답 본문
json()?arrayJSON 디코딩 (실패 시 null)
ok()bool200~299 여부
headers()array응답 헤더
기본 사용법
// GET 요청
$res = http()->get('https://api.example.com/users');
$users = $res->json();

// JSON POST
$res = http()->header('Authorization', 'Bearer ' . $token)
    ->json('https://api.example.com/posts', [
        'title' => 'Hello',
        'body'  => 'World',
    ]);

// 파일 다운로드
http()->download('https://example.com/file.zip', '/tmp/file.zip');
Base URL + 타임아웃
// API 클라이언트 패턴
$api = http()->base('https://api.example.com')
    ->header('X-API-Key', $key)
    ->timeout(10);

$users = $api->get('/users')->json();
$posts = $api->get('/posts')->json();
팁: timeout() 기본값은 30초입니다. 외부 API 호출 시 적절한 타임아웃을 설정하세요. 응답 객체의 ->status(), ->json(), ->body()로 결과를 확인할 수 있습니다.
