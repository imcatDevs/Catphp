Cat\Flash — 플래시 메시지
flash()

세션 기반 일회성 메시지입니다. 폼 제출 후 리다이렉트 시 성공/에러 메시지를 전달하는 PRG(Post-Redirect-Get) 패턴에 사용합니다.

메시지는 한 번 읽으면 자동으로 삭제됩니다. success(), error(), warning(), info()로 타입별 메시지를 간편하게 설정할 수 있습니다.

전체 메서드 레퍼런스
메서드반환설명
set(string $type, string $message)void타입별 메시지 저장 (Guard 살균 적용)
get()array전체 메시지 읽기 (읽은 후 삭제)
has()bool메시지 존재 여부
success(string $msg)void성공 메시지
error(string $msg)void에러 메시지
warning(string $msg)void경고 메시지
info(string $msg)void정보 메시지
PRG 패턴 예제
// POST 핸들러 — 처리 후 리다이렉트
router()->post('/settings', function() {
    // 설정 저장 로직...
    flash()->success('설정이 저장되었습니다.');
    header('Location: /settings');
});

// GET 핸들러 — 뷰에서 메시지 표시
if (flash()->has()) {
    foreach (flash()->get() as $msg) {
        echo '<div class="alert alert--' . $msg['type'] . '">' . $msg['message'] . '</div>';
    }
}
PRG 패턴: POST 처리 후 flash()->success('저장됨') → 리다이렉트 → GET에서 flash()->get()로 전체 메시지 배열을 읽습니다. 메시지는 한 번 읽으면 자동 삭제됩니다.
관련: Router
