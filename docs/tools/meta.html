Cat\Meta — SEO 메타 태그
meta()

SEO 메타 태그를 체이닝으로 관리합니다. title, description, canonical, Open Graph, Twitter Card, JSON-LD 구조화 데이터를 하나의 도구로 설정합니다.

render()로 모든 메타 태그를 HTML로 출력하고, sitemap()으로 XML 사이트맵을 생성합니다. reset()으로 페이지별 메타 데이터를 초기화할 수 있습니다 (뮤터블 구조).

전체 메서드 레퍼런스
메서드반환설명
title(string $t)self페이지 타이틀
description(string $d)self메타 설명
canonical(string $url)self정규 URL
og(string $prop, string $val)selfOpen Graph 태그
twitter(string $prop, string $val)selfTwitter Card 태그
jsonLd(array $data)selfJSON-LD 구조화 데이터
reset()self모든 메타 데이터 초기화
render()string메타 태그 HTML 문자열 반환
sitemap(array $urls)stringXML 사이트맵 생성
사용 예제
// 페이지별 SEO 설정
meta()->title('CatPHP — 경량 PHP 프레임워크')
    ->description('PHP 8.2+ 전용 경량 프레임워크')
    ->canonical('https://catphp.dev')
    ->og('image', 'https://catphp.dev/og.png')
    ->twitter('card', 'summary_large_image');

// <head> 안에서 출력
echo meta()->render();

// 사이트맵 생성
$xml = meta()->sitemap([
    ['loc' => 'https://catphp.dev', 'priority' => '1.0'],
    ['loc' => 'https://catphp.dev/docs', 'priority' => '0.8'],
]);
뮤터블: Meta는 다른 도구와 달리 $this 기반입니다. 페이지별로 reset()을 호출하고, render()를 <head> 안에 배치하세요.
