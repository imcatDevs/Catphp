Cat\Event — 이벤트 시스템
event()

Pub/Sub 이벤트 시스템입니다. 리스너를 등록(on/once)하고 이벤트를 발행(emit)하면 등록된 모든 리스너가 순서대로 실행됩니다.

on()/once()는 리스너 ID를 반환하며, off()로 특정 리스너만 제거할 수 있습니다. once()는 한 번 실행 후 자동 제거됩니다.

전체 메서드 레퍼런스
메서드반환설명
on(string $event, callable $fn)int리스너 등록 (ID 반환)
once(string $event, callable $fn)int1회용 리스너 등록
emit(string $event, mixed ...$args)self이벤트 발행 (모든 리스너 실행)
hasListeners(string $event)bool리스너 존재 여부
off(string $event, ?int $id = null)self리스너 제거 (ID 지정 시 특정, null 시 전체)
기본 사용법
// 리스너 등록
$id = event()->on('user.created', function(array $user) {
    logger()->info('회원가입', $user);
});

event()->on('user.created', function(array $user) {
    mailer()->to($user['email'])->send('환영합니다!');
});

// 이벤트 발행 — 모든 리스너 실행
event()->emit('user.created', ['id' => 1, 'email' => 'cat@php.dev']);

// 리스너 제거
event()->off('user.created', $id);
활용: 회원가입 후 이메일 발송, 주문 완료 후 알림, 게시글 작성 후 캐시 무효화 등 비즈니스 로직을 분리하여 코드 결합도를 낮출 수 있습니다.
