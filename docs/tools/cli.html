Cat\Cli — CLI 명령어 프레임워크
cli()

CLI(명령줄) 실행 환경을 위한 도구입니다. 터미널 색상 출력, 테이블 렌더링, 진행률 표시, 사용자 입력 프롬프트 등 CLI 애플리케이션에 필요한 기능을 제공합니다.

cli.php에서 커스텀 명령어를 등록하고, php cli.php command:name 형식으로 실행합니다. $argv null 방어가 적용되어 안전합니다.

전체 메서드 레퍼런스
메서드반환설명
command(string $name, string $desc, callable $fn)self명령어 등록 (이름, 설명, 핸들러)
group(string $prefix, callable $fn)self명령어 그룹 (네임스페이스)
run()void$argv 기반 명령어 실행
arg(int $index)?string위치 인자 (0-indexed)
option(string $name, mixed $default = null)mixed옵션 파싱 (--key=value)
confirm(string $q, ?bool $default = null)bool확인 질문 (y/n)
prompt(string $q, ?string $default = null)string텍스트 입력 프롬프트
choice(string $q, array $options)?string선택지 메뉴
info/success/warn/error(string $msg)void색상 메시지 출력
table(array $headers, array $rows)void테이블 출력
progress(int $current, int $total)void프로그레스 바
명령어 등록
// cli.php — 커스텀 명령어 등록 (이름, 설명, 핸들러)
cli()->command('cache:clear', '캐시 전체 삭제', function() {
    cache()->clear();
    cli()->success('캐시가 삭제되었습니다.');
});

cli()->command('user:list', '사용자 목록 출력', function() {
    $users = db()->table('users')->all();
    cli()->table(['ID', 'Name', 'Email'], $users);
});

cli()->run();
인자/옵션 파싱 + 인터랙티브
// php cli.php firewall:ban 203.0.113.50 --ttl=3600
cli()->command('firewall:ban', 'IP 차단', function() {
    $ip  = cli()->arg(0) ?? cli()->prompt('차단할 IP');
    $ttl = (int) cli()->option('ttl', 0);

    if (cli()->confirm("{$ip} 차단?")) {
        firewall()->ban($ip);
        cli()->success("{$ip} 차단 완료");
    }
});
실행
# 프로젝트 초기화 (전체 도구 기본 config 생성 + 보안 키 + 디렉토리)
php cli.php config:init

# 보안 키 재생성
php cli.php key:generate

# 인터랙티브 설정 마법사
php cli.php setup

# 환경 진단
php cli.php check:env

# 일반 명령어
php cli.php cache:clear
php cli.php user:list
php cli.php firewall:ban 203.0.113.50 --ttl=3600
팁: 인자 없이 php cli.php을 실행하면 등록된 모든 명령어와 설명이 출력됩니다.
