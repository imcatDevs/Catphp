Schedule

Cat\Schedule — Cron 스케줄러
schedule()

Cron 기반 태스크 스케줄러입니다. CLI 명령어나 콜백을 정해진 주기로 자동 실행합니다.

시스템 crontab에 * * * * * php cli.php schedule:run 한 줄만 등록하면, 나머지 스케줄은 PHP에서 관리합니다.

주요 메서드
메서드반환설명
command(string $cmd)selfCLI 명령어 예약
call(callable $cb, string $desc)self콜백 함수 예약
cron(string $expression)self원시 cron 표현식
everyMinute / everyFiveMinutes / hourlyself주기 헬퍼
daily / dailyAt(string $time)self매일 / 지정 시각
weekly / weeklyOn(int $day, string $time)self매주 / 지정 요일
monthly / monthlyOn(int $day, string $time)self매월 / 지정일
withoutOverlapping()self중복 실행 방지
description(string $desc)self태스크 설명
run()int현재 시각 기준 실행
list()array등록된 태스크 목록
사용 예제
// cli.php — 스케줄 등록
schedule()->command('cache:clear')->daily()
    ->description('매일 캐시 정리');

schedule()->call(function() {
    db()->table('sessions')->where('expires_at', '<', date('Y-m-d'))->delete();
}, '만료 세션 정리')->hourly()->withoutOverlapping();

schedule()->command('queue:work --max=100')
    ->everyMinute()->withoutOverlapping();
Crontab 설정
# 시스템 crontab에 한 줄만 등록
* * * * * cd /path/to/project && php cli.php schedule:run >> /dev/null 2>&1

# 등록된 스케줄 확인
php cli.php schedule:list
팁: withoutOverlapping()은 락 파일로 중복 실행을 방지합니다. 장시간 실행되는 태스크에 반드시 적용하세요.
관련: Cli Queue
