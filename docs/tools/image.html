Cat\Image — 이미지 처리
image()

GD 기반 이미지 처리 도구입니다. 리사이즈, 크롭, 워터마크(텍스트/이미지), 형식 변환을 체이닝 API로 제공합니다.

JPEG, PNG, GIF, WebP 형식을 지원하며, 워터마크는 TrueType 폰트를 사용할 수 있습니다. width()/height() 게터로 현재 이미지 크기를 확인할 수 있습니다.

전체 메서드 레퍼런스
메서드반환설명
open(string $path)self이미지 파일 열기
resize(int $w, int $h)self리사이즈 (직접 크기 지정)
thumbnail(int $maxW, int $maxH)self비율 유지 썸네일
crop(int $x, int $y, int $w, int $h)self크롭
watermark(string $text, int $x, int $y, int $fontSize, ?string $fontPath)self텍스트 워터마크 (TrueType 지원)
save(string $path)bool파일로 저장 (확장자로 형식 자동 감지)
convert(string $outputPath)bool포맷 변환 (save 래퍼)
width()int현재 너비
height()int현재 높이
사용 예제
// 썸네일 생성 (비율 유지)
image()->open('uploads/photo.jpg')
    ->thumbnail(300, 300)
    ->save('uploads/thumb_photo.jpg');

// 워터마크 추가 + WebP 변환
image()->open('uploads/photo.jpg')
    ->watermark('© CatPHP', 10, 10, 16)
    ->save('uploads/photo.webp');
지원 형식: JPEG, PNG, GIF, WebP. save()에서 확장자로 형식을 자동 결정합니다. WebP 변환 시 용량이 30~50% 절감됩니다.
관련: Upload
