Cat\Sitemap — XML 사이트맵 생성
sitemap()

XML Sitemap 생성 도구입니다. 개별 URL 추가, DB 쿼리 결과 자동 변환, 사이트맵 인덱스 생성을 지원합니다.

changefreq는 always|hourly|daily|weekly|monthly|yearly|never만 허용되며, priority는 0.0~1.0 범위로 자동 클램핑됩니다. 파일당 최대 50,000개 URL (사이트맵 스펙).

전체 메서드 레퍼런스
메서드반환설명
url(string $loc, ?string $lastmod, ?string $changefreq, ?float $priority)selfURL 엔트리 추가
urls(array $urls)self복수 URL 일괄 추가
fromQuery(array $rows, string $urlPattern, ...)selfDB 결과 → URL 자동 생성
index(array $sitemaps)self사이트맵 인덱스 생성
render()stringXML 문자열 반환
output()neverContent-Type + echo + exit
save(string $path)bool파일로 저장
count()intURL/사이트맵 수
기본 사용법
// 수동 URL 추가 + 파일 저장
sitemap()->url('/', '2024-01-01', 'daily', 1.0)
    ->url('/about', '2024-01-01', 'monthly', 0.8)
    ->url('/contact', null, 'yearly', 0.5)
    ->save('Public/sitemap.xml');
DB 연동 — fromQuery()
// DB 결과를 자동으로 사이트맵 URL로 변환
$posts = db()->table('posts')
    ->where('status', 'published')
    ->orderBy('updated_at', 'DESC')->all();

sitemap()->url('/', null, 'daily', 1.0)
    ->fromQuery($posts, '/post/{slug}', 'updated_at')
    ->output();  // Content-Type: application/xml + echo + exit
사이트맵 인덱스
// 여러 사이트맵을 인덱스로 묶기
sitemap()->index([
    '/sitemap-posts.xml',
    '/sitemap-pages.xml',
    ['loc' => '/sitemap-products.xml', 'lastmod' => '2024-06-01'],
])->save('Public/sitemap.xml');
CLI 명령어
# DB 테이블에서 사이트맵 자동 생성
php cli.php sitemap:generate --table=posts --url=/post/{slug}
팁: config('sitemap.base_url')로 기본 URL을 설정하면 상대 경로가 자동으로 절대 URL로 변환됩니다. 캐시 연동 시 cache()->remember('sitemap', fn() => sitemap()->...->render(), 3600) 패턴을 사용하세요.
