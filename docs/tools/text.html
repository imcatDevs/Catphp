Cat\Text — 텍스트 유틸
text()

텍스트 처리 유틸리티 모음입니다. 발췌문 생성, 읽기 시간 계산, 단어/문자 수 카운트, HTML 태그 제거, 텍스트 자르기 등 블로그·CMS에서 자주 사용하는 기능을 제공합니다.

전체 메서드 레퍼런스
메서드반환설명
excerpt(string $text, int $len = 200)string발췌문 생성 (HTML 제거, 단어 경계 자르기)
readingTime(string $text)string읽기 시간 ("N분" 형태). 한국어 500자/분 기준
wordCount(string $text)int단어 수
charCount(string $text, bool $includeSpaces)int문자 수 (공백 포함 여부 선택)
stripTags(string $text)stringHTML 태그 제거
truncate(string $text, int $len, string $end)string텍스트 자르기 (말줄임 추가)
사용 예제
$html = '<p>CatPHP는 PHP 8.2+ 전용 <strong>경량 프레임워크</strong>입니다...';

// 발췌문 (100자)
text()->excerpt($html, 100);
// → "CatPHP는 PHP 8.2+ 전용 경량 프레임워크입니다..."

// 읽기 시간
text()->readingTime($longArticle);  // → "5분"

// 단어/문자 수
text()->wordCount($body);   // → 1234
text()->charCount($body);   // → 5678

// HTML 태그 제거
text()->stripTags($html);
팁: readingTime()은 한국어(CJK) 기준 분당 500자, 영어 기준 분당 200단어로 계산합니다. excerpt()는 단어 중간을 자르지 않고 공백 경계에서 잘라냅니다.
