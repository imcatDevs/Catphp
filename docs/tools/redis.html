Redis

Cat\Redis — ext-redis 클라이언트
redis()

ext-redis 래퍼입니다. String, Hash, List, Set, Sorted Set, Pub/Sub 명령어를 지원하며, remember() 패턴으로 캐시로도 활용됩니다.

config/app.php의 redis 키에서 호스트, 포트, 비밀번호, DB 인덱스를 설정합니다. 연결은 지연 초기화되어 실제 사용 시점에만 연결됩니다.

설정 — config/app.php
'redis' => [
    'host'     => '127.0.0.1',
    'port'     => 6379,
    'password' => null,
    'database' => 0,
]
주요 메서드
메서드반환설명
set(string $key, mixed $value, int $ttl = 0)bool값 설정 (TTL 초, 0=영구)
get(string $key, mixed $default = null)mixed값 읽기
del(string ...$keys)int키 삭제
exists(string $key)bool키 존재 여부
incr(string $key, int $by = 1)int증가
remember(string $key, callable $cb, int $ttl)mixed캐시 기억 패턴
hSet / hGet / hGetAll / hDel—Hash 명령어
rPush / lPop / blPop / lLen / lRange—List 명령어 (큐)
sAdd / sMembers / sIsMember / sRem—Set 명령어
zAdd / zRange / zRem / zScore—Sorted Set
publish(string $channel, mixed $msg)intPub/Sub 메시지 발행
raw()\Redis원시 인스턴스 반환
기본 사용법
// String 캐시
redis()->set('user:1:name', '김개발', 3600);
$name = redis()->get('user:1:name');

// remember 패턴
$stats = redis()->remember('dashboard:stats', fn() =>
    db()->table('users')->count()
, 300);

// 카운터
redis()->incr('page:views');
요구사항: ext-redis PHP 확장이 필요합니다. php -m | grep redis로 확인하세요.
