Cat\Notify — 다채널 알림
notify()

다채널 알림 시스템입니다. 이메일, 텔레그램, 슬랙 등 여러 채널로 동시에 알림을 발송합니다.

via()로 채널을 선택하고, send()로 일괄 발송합니다. 채널을 지정하지 않으면 설정된 기본 채널로 발송됩니다.

전체 메서드 레퍼런스
메서드반환설명
to(string ...$recipients)self수신자 설정
via(string ...$channels)self채널 선택 (mail, telegram, slack 등)
send(string $subject, string $body)array발송 (채널별 성공 여부 반환)
alert(string $message)array간편 발송 (기본 채널)
사용 예제
// 여러 채널로 동시 발송
notify()
    ->to('admin@example.com')
    ->via('mail', 'telegram')
    ->send('서버 경고', '디스크 사용량 90% 초과');

// 간편 알림 (기본 채널)
notify()->alert('배포 완료: v2.1.0');
연동: Mail, Telegram 도구가 내부적으로 사용됩니다. 각 채널의 설정이 config/app.php에 올바르게 입력되어야 합니다.
관련: Mail Telegram
