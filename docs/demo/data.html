check_circle Valid
Cat\Valid

규칙 기반 입력 검증. required, email, min, max, between, regex, nullable 등 체이닝

$v = valid()->rules([
    'name'  => 'required|min:2|max:50',
    'email' => 'required|email',
    'age'   => 'required|numeric|min:1|max:120',
    'bio'   => 'nullable|max:500',
])->check($data);

if ($v->fails()) {
    json()->fail('검증 실패', $v->errors(), 422);
}

cloud_upload Upload
Cat\Upload

파일 업로드 처리. 확장자/MIME/크기 검증, finfo 교차 확인, 안전한 파일명 생성

// 기본 업로드
$file = upload()->file('avatar')
    ->maxSize('2M')
    ->extensions(['jpg', 'png', 'webp'])
    ->save('storage/avatars');

// 결과
$file->path;     // storage/avatars/abc123.jpg
$file->name;     // abc123.jpg
$file->size;     // 102400
$file->mime;     // image/jpeg

// 다중 업로드
$files = upload()->files('photos')
    ->maxSize('5M')->saveAll('storage/photos');

last_page Paginate
Cat\Paginate

쿼리 결과 페이지네이션. 자동 offset 계산, 윈도우 트렁케이션 링크, toArray() JSON 직렬화

$p = paginate()->total(150)->perPage(10)->page(7);
$p->offset();    // 60
$p->lastPage();  // 15
$p->toArray();   // JSON 직렬화

// DB 연동
$users = db()->table('users')
    ->limit($p->perPage())
    ->offset($p->offset())
    ->all();

cookie Cookie
Cat\Cookie

보안 쿠키 관리. Encrypt 연동 자동 암호화/복호화, HttpOnly + Secure + SameSite 기본 적용

// 쿠키 설정 (7일)
cookie()->set('locale', 'ko', 604800);
cookie()->set('theme', 'dark', 2592000);

// 쿠키 읽기 (자동 복호화)
$locale = cookie()->get('locale', 'en');

// 삭제 / 전체 확인
cookie()->del('locale');
$has = cookie()->has('theme');
